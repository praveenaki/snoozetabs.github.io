# Google Drive Sync (PRO)

Back up and sync your snoozes across devices using Google Drive.

## What Syncs

- All snoozes (sleeping and saved)
- Workspaces
- Settings

**Does not sync:**
- License key (stored separately)
- Local history

## Setting Up Sync

1. Go to **Settings**
2. Find **Google Drive Sync** section
3. Click **"Connect Google Account"**
4. Sign in and grant permissions
5. Toggle **"Sync enabled"** ON

## How Sync Works

### Automatic Sync
- Syncs when you make changes
- Syncs on extension startup
- Syncs periodically in background

### Manual Sync
Click **"Sync Now"** to force an immediate sync.

### Sync Status
Shows last successful sync time.

## Multi-Device Setup

1. Install Snooze Tabs on all devices
2. Activate PRO license on each device
3. Connect same Google account on each
4. Enable sync on each device

**Important:** Use the same Google account on all devices.

## Conflict Resolution

If the same snooze is modified on multiple devices:
- Most recent change wins
- Sync happens quickly enough that conflicts are rare

## Disconnecting

1. Go to **Settings**
2. Click **"Disconnect"**
3. Sync stops, local data remains

## Privacy

- Data stored in your personal Google Drive
- Only Snooze Tabs can access it
- Encrypted in transit
- You can delete the data anytime from Drive

## Troubleshooting

### Sync not working
1. Check internet connection
2. Verify Google account is connected
3. Click "Sync Now" to force sync
4. Check Google Drive storage quota

### Data missing on new device
1. Ensure PRO is activated
2. Connect same Google account
3. Enable sync
4. Wait for initial sync or click "Sync Now"
