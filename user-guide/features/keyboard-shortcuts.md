# Keyboard Shortcuts

Snooze tabs without opening the popup using keyboard shortcuts.

## Available Shortcuts

| Shortcut (Windows/Linux) | Shortcut (Mac) | Action |
|--------------------------|----------------|--------|
| `Alt+Shift+S` | `Ctrl+Shift+S` | Later Today |
| `Alt+Shift+E` | `Ctrl+Shift+E` | This Evening |
| `Alt+Shift+T` | `Ctrl+Shift+T` | Tomorrow |
| `Alt+Shift+W` | `Ctrl+Shift+W` | This Weekend |

## How They Work

1. Navigate to the tab you want to snooze
2. Press the keyboard shortcut
3. Tab closes immediately
4. Snooze is created with the preset time

## Customizing Shortcuts

Chrome allows you to customize extension shortcuts:

1. Go to `chrome://extensions/shortcuts`
2. Find "Snooze Tabs"
3. Click the pencil icon next to any shortcut
4. Press your preferred key combination
5. Click OK

### Tips for Custom Shortcuts
- Avoid conflicts with browser shortcuts
- Use memorable combinations
- Consider your keyboard layout

## Best Practices

### Batch Processing
When clearing many tabs:
1. `Ctrl+Tab` to move to next tab
2. `Alt+Shift+T` to snooze to tomorrow
3. Repeat

### Quick Triage
- `Alt+Shift+E` for "deal with tonight"
- `Alt+Shift+T` for "deal with tomorrow"
- `Alt+Shift+W` for "weekend reading"

## Limitations

Shortcuts won't work on:
- `chrome://` pages
- `chrome-extension://` pages
- Other browser internal pages

You'll see a brief error notification if you try.
